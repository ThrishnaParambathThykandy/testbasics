What is Agile?

Agile is a software development methodology that focuses on delivering small, incremental pieces of working software frequently.

Agile encourages collaboration, flexibility, and quick feedback throughout the development lifecycle.

Agile Testing

In Agile, testing is performed continuously alongside development. Testers are involved from the beginning of the sprint.

Testing activities are not a separate phase but part of every sprint.

Software Testing Life Cycle (STLC)

  • Requirement Analysis
  • Test Planning
  • Test Case Design
  • Test Environment Setup
  • Test Execution
  • Test Closure

  • Sprint Review: Demo completed work
  • Sprint Retrospective: Improve the process

Roles in Agile

  • Product Owner
  • Scrum Master
  • Development Team
  • QA / Tester
